## 🧠 Концепция

**WakeSmart** — это не просто будильник. Это система, которая заботится о вашем пробуждении:

- ✅ **Постепенное увеличение громкости** — звук нарастает от 0% до 100% за 5–10 минут.
- ✅ **Имитация рассвета** — плавное увеличение яркости экрана или внешней лампы (через умные устройства).
- ✅ **Выбор дней недели** — будильник срабатывает только в заданные дни.
- ✅ **Отложенный сон (Snooze)** — с настраиваемым интервалом (5–30 минут).
- ✅ **Кастомные мелодии** — поддержка MP3/WAV/OGG или встроенных сигналов.
- ✅ **Интеграция с прогнозом погоды** — рекомендация оптимального времени пробуждения (опционально).
- ✅ **Сохранение настроек** — все параметры хранятся в локальном JSON / SQLite.
- ✅ **Голосовое оповещение** (в некоторых версиях) — произносит время и погоду.

## 🚀 Функциональность

| Команда / Кнопка        | Описание |
|-------------------------|----------|
| `set <HH:MM> [days]`    | Установить будильник на время, дни (например, `set 07:30 mon,wed,fri`) |
| `list`                  | Показать все установленные будильники |
| `remove <id>`           | Удалить будильник по ID |
| `snooze <minutes>`      | Отложить текущий будильник на N минут |
| `stop`                  | Остановить звонок |
| `volume <0-100>`        | Установить максимальную громкость |
| `gradual <minutes>`     | Установить время плавного нарастания (по умолчанию 5 мин) |
| `melody <file>`         | Установить путь к мелодии |
| `light <on/off>`        | Включить/выключить имитацию рассвета |
| `exit`                  | Сохранить настройки и выйти |

## 📦 Содержимое репозитория

| Файл                | Язык       | Особенности |
|---------------------|------------|-------------|
| `alarm_python.py`   | Python     | + Tkinter GUI, `playsound`, имитация света через яркость экрана |

## 🛠️ Как запустить

Каждая версия самодостаточна и имеет свои зависимости. Установите необходимое окружение и запустите:

```bash
# Python
pip install playsound
python alarm_python.py

# C++ (Qt)
qmake && make
./alarm_cpp

# Java
javac alarm_java.java && java alarm_java

# C# (WPF)
dotnet run

# Go
go run alarm_go.go

# Rust
cargo build --release && ./target/release/alarm_rs

# Node.js (Electron)
npm install
npm start
